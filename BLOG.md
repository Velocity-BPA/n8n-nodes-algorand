# Automate Your Algorand Blockchain Operations with n8n

We're excited to announce the release of n8n-nodes-algorand, a comprehensive community node that brings Algorand blockchain functionality directly into your n8n workflows!

## The Challenge

Blockchain integration has traditionally required extensive development expertise. Whether you're managing ASA tokens, minting NFTs, deploying smart contracts, or simply monitoring transactions, you've likely needed custom code and complex API integrations. For teams wanting to incorporate Algorand blockchain operations into their business processes, this technical barrier has been a significant obstacle.

## Introducing n8n-nodes-algorand

Velocity BPA has created a powerful solution that democratizes Algorand blockchain automation. Our new community node enables you to interact with the Algorand blockchain using n8n's intuitive visual workflow builder—no coding required.

## Key Features

This node provides comprehensive Algorand functionality:

- **Account Management**: Check balances, monitor account activity, and manage multiple accounts
- **Transaction Operations**: Send payments, track transaction status, and automate payment workflows
- **ASA Token Support**: Create, transfer, and manage Algorand Standard Assets
- **NFT Capabilities**: Mint, transfer, and manage NFTs on Algorand
- **Smart Contract Integration**: Interact with Algorand smart contracts and application calls

Whether you're building automated treasury management systems, NFT minting platforms, or blockchain-based notification systems, this node provides the building blocks you need.

## Getting Started

Installation is straightforward. In your n8n instance, run:


npm install n8n-nodes-algorand


After restarting n8n, you'll find the Algorand node available in your node palette, ready to integrate into your workflows.
